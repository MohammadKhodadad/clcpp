CORE RULES:
- It must be a question: phrase it as an interrogative ("Which...?", "What...?", "What is the...?") that ends with "?". A description or noun phrase is invalid — rewrite it.
- Answer is the concept: the one and only natural answer to your question must be this concept.
- Never name the concept: the question must never contain the concept's name or ANY of its listed aliases — in any language or spelling. If the name would appear, rewrite the question.
- Describe, don't name: build the question from the concept's grounded technical attributes as stated in the passages — its structure, functional groups, role or use, properties, the reaction or composition it takes part in, or characteristic parameters.
- Full direct support: every distinguishing detail you use must be explicitly present in a passage about this concept. No inference, no outside knowledge.
- Numerical fidelity: reproduce any numbers, units, and ranges exactly as they appear (e.g., "85–87°C", not "about 86°C").
- Single answer: make the question specific enough that THIS concept — not a chemically similar one — is the natural answer.
- Paraphrase, don't lift: technical terms (e.g., "Pd/C catalyst", "underground aquifer") may be reused for clarity, but do not lift long descriptive phrases or sentence fragments from the source as the question scaffold — rewrite those in your own words.
- Natural wording: write it the way a researcher would actually ask — short, specific, and direct. No patent-summary language and no document phrases ("described in the invention", "according to the text").
- Standalone query for a search engine (no document references): this question is a query for an Information Retrieval (IR) system — the person typing it has NOT seen this document and is searching to FIND it. Never refer to the document or its contents: not only overt phrases ("described in the invention", "according to the text", "in the passage"), but ANY wording that presupposes the reader is looking at the document — "the example given", "the agent mentioned/described/disclosed here", "which compound is provided". Write a self-contained question. Bad: "Which agent example is given that increases ROS?" → Good: "Which agent increases ROS?"

BUILD THE QUESTION FROM THE MOST DISTINGUISHING OF:
  1. Structure: functional groups, scaffold, substituents, salt/ester form, molecular formula
  2. Role / use: what the concept does or is used for in the passage
  3. Property: a characteristic physical or chemical property
  4. Reaction / process: the reaction, synthesis step, or composition it takes part in
  5. Parameter: a characteristic condition or value tied to it
