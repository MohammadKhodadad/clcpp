# CLCPP

This repository contains a clean export of the multi-lingual chemical QAC algorithm and the two dataset snapshots used with it.

## Layout

- `code/` contains the Python implementation.
- `data/` contains downloaded parquet files from the two Hugging Face datasets:
  - included snapshot: data/multi-lingual-qac-chem-patents
  - included snapshot: data/multi-lingual-qac-epo
